# 똥싸당 운세 - Claude 프로젝트 부트스트랩

## 프로젝트 소개

이 프로젝트는 "똥싸당 운세"라는 모바일 웹 기반 슬롯 운세 게임이다.

사용자는 로그인 후 슬롯을 돌리고 운세 카드를 획득한다.

프로젝트 목표는 단순 운세 확인 서비스가 아니라

"매일 접속해서 스핀을 돌리고 싶은 운세 수집형 게임"

을 만드는 것이다.
