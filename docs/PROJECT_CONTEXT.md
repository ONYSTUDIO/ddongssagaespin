# 똥싸당 운세 (DdongssagaeSpin)

## 프로젝트 개요

똥싸당 운세는 모바일 우선(Mobile First) HTML5 슬롯 기반 운세 게임이다.

사용자는 로그인 후 스핀을 돌리고, 결과에 따라 운세를 확인할 수 있다.

프로젝트 목표는 다음과 같다.

* AI 기반 1인 게임 개발 학습
* 실제 서비스 가능한 웹 게임 제작
* 추후 사업화 및 광고 수익화
* Node.js 기반 서버 구축
* 사용자 데이터 수집 및 통계 기능 제공
