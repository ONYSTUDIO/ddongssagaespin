# 똥싸당 운세 - 게임 규칙 문서

## 게임 개요

똥싸당 운세는 3릴 슬롯 기반 운세 게임이다.

사용자는 스핀을 돌려 결과를 확인하고,
당첨 시 운세 카드를 획득한다.

---

# 슬롯 구조

## 릴 수

3 Reel

## 판정 라인

가운데 라인만 판정

예시

[코기]
[코기]
[코기]

당첨

---

# 포춘쿠키 시스템

포춘쿠키는 별도 메타 콘텐츠

사용자는 포춘쿠키를 열어
추가 메시지를 확인 가능

포춘 쿠키 메세지를 작성하는 기능

---

# 랭킹 시스템

## 기준

일일 랭킹

매일 초기화

---

## 계산 방식

획득한 행운 점수 중 가장 높은 점수

높은 점수 순 정렬

# 게임 목표

사용자가 단순히 하루 운세를 확인하고 떠나는 것이 아니라

매일 접속하여

* 스핀을 돌리고
* 행운 점수를 모으고
* 랭킹에 도전하고
* 운세 카드를 수집하는

운세 수집형 슬롯 게임을 목표로 한다.
